# 📱 Lab1 – React Native Mobile App (Expo)

Навчальний мобільний застосунок, створений на **React Native + Expo**.  
Проєкт містить навігацію між трьома екранами з використанням **Material Top Tabs**:

## 🧭 Екрани застосунку
- **Головна** – список новин (зображення, заголовок, дата, короткий опис)
- **Фотогалерея** – сітка зображень (по 2 в рядок, 10 на екран)
- **Профіль** – форма реєстрації:
    - Email
    - Пароль
    - Повтор пароля
    - Прізвище
    - Ім'я
    - Кнопка "Зареєструватися"

---

# 🚀 Інструкція із запуску

## 1️⃣ Встановлення залежностей
У корені проєкту:

```bash
npm install
```

## 2️⃣ Запуск проєкту
npx expo start
▶️ Способи запуску мобільного додатка
📲 1. Запуск на фізичному телефоні (Expo Go)

Призначення: швидке тестування на реальному пристрої

Як:

Встановити Expo Go з Play Market / App Store

Запустити npx expo start

Відсканувати QR-код камерою або через Expo Go

Особливості:

не потребує Android Studio / Xcode

швидкий старт

обмежений доступ до native API

🤖 2. Android Emulator

Призначення: тестування без фізичного пристрою

Як:

npx expo start --android

```bash
npx expo start --android
```
Особливості:

потребує Android Studio + Emulator

стабільне середовище

повний контроль тестування UI

🍎 3. iOS Simulator (macOS)

Призначення: тестування iOS-версії

```bash
npx expo start --ios
```
Особливості:

працює тільки на macOS

потребує Xcode

точна симуляція iOS-пристроїв
🌐 4. Web-режим

Призначення: швидка перевірка логіки в браузері

```bash
npx expo start --web
```
Особливості:

не є повноцінною мобільною емуляцією

використовується для логіки та UI-тестів
